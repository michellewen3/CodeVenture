About the Game

The core value of the game is to provide individuals with an innovative, engaging, and beginner-friendly way to start programming through taking them on a journey involving an intriguing and unforgettable storyline. We developed this game to encourage high school students, specifically minorities who may have limited access to learning resources, to start learning code through showing them how fun and rewarding programming can be. Moreover, we hope to fill the diversity gap in computer science by ensuring all students have equal access to learning opportunities through using our application.
